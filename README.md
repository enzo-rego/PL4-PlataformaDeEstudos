PlataformaDeEstudos

Criar uma plataforma de estudos funcionais que agrupe estudantes e professores qualificados, com aulas gravadas e questões de simulados focados na preparação para vestibulares. 

Tecnologias:Fórum de Dúvidas, Videoaulas Gravadas, Simulados Disponíveis, Explicações interativas, Apostilas Teóricas.
